To build and install VPlants modules, type: 

  python setup.py install

If you want to build several modules at once, use the script
multisetup.py :

  python multisetup.py [targets]

where target are the same that setup.py
  
  
Nota Bene: 
  Specific system options can be set both from the command line, or in a file "options.py".
  Read SConsX documentation for further explanations.
